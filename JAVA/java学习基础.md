# 第一部分：JavaSE基础 javaEE javaME
## 1. Java语言概述
### 1.1 Java简介

- 定义：Java是一种面向对象的、跨平台的编程语言

- 特点：

    简单易学   
    面向对象  
    平台无关性（一次编写，到处运行）  
    安全性高  
    多线程支持  
    自动垃圾回收

### 1.2 Java程序运行机制
‘’‘
Java源代码(.java) → 编译器 → 字节码(.class) → JVM → 机器码 → 运行
’‘’

### 1.3 JDK、JRE、JVM的关系
- JVM（Java Virtual Machine）：Java虚拟机，运行Java字节码
- JRE（Java Runtime Environment）：Java运行环境 = JVM + 核心类库
- JDK（Java Development Kit）：Java开发工具包 = JRE + 开发工具

## 2. Java基础语法
### 2.1 数据类型
#### 2.1.1 什么是数据类型？
数据类型的通俗理解：  
- 数据类型就像盒子的类型，决定了能装什么、装多少  
- 就像生活中，装水的容器和装书的容器是不同的，数据类型也决定了数据的性质和大小

** 为什么要定义数据类型？ **
- 告诉计算机这个数据是什么类型、占用多少内存空间
- 就像购物时，买鸡蛋用鸡蛋盒，买苹果用水果盒，不同类型的东西用不同的容器

** 数据类型的分类：**
- 基本数据类型：8种，用来存储简单的数值，就像直接存放的物品
- 引用数据类型：类、数组、字符串等，用来存储复杂的数据，就像存放物品的地址

#### 2.1.2 基本数据类型（8种）
**基本数据类型的特点：**
- 直接存储值，不是地址
- 有固定的大小和取值范围
- 8种基本类型，分为4大类
